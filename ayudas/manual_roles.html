Introducción

El módulo de Asignación de Roles a Usuarios es una herramienta intuitiva y visual que permite a los administradores gestionar los roles asignados a los usuarios del sistema de manera rápida y eficiente.

Interfaz Drag & Drop

Arrastra y suelta roles sobre usuarios para asignarlos de forma inmediata, sin necesidad de formularios complejos.

Actualización Instantánea

Los cambios se aplican inmediatamente en la base de datos y se reflejan en tiempo real en la interfaz.

Control de Acceso

Solo usuarios con Rol 0 (Administrador) pueden acceder a este módulo para garantizar la seguridad.

Notificaciones Toast

Recibe confirmaciones visuales de cada acción realizada mediante notificaciones emergentes elegantes.

Acceso al Sistema

Requisito de Permisos

Solo los usuarios con Rol 0 (Administrador) pueden acceder a este módulo. Si no tienes los permisos necesarios, serás redirigido automáticamente.

Pasos para Acceder:

1

Iniciar Sesión

Ingresa al sistema con tus credenciales de administrador (usuario con Rol 0).

2

Navegar al Módulo

Desde el menú principal, selecciona la opción "Asignación de Roles" o accede directamente a rol.html.

3

Verificación Automática

El sistema verificará automáticamente tu token de autenticación y tus permisos antes de cargar la interfaz.

Descripción de la Interfaz

La interfaz está dividida en tres secciones principales:

1. Barra Superior (Header)

  • Icono de Roles: Imagen visual que identifica el módulo (asignarol.png).
  • Título: "Asignación de Roles" - Indica claramente la función del módulo.
  • Botón Salir: Ubicado en la esquina superior derecha, permite regresar a la página anterior o al layout principal.

2. Panel Izquierdo - Roles Disponibles

Este panel muestra todos los roles que puedes asignar a los usuarios:

  • Roles con Colores: Cada rol tiene un color distintivo para facilitar su identificación visual:
    • Rol 1 - Azul
    • Rol 2 - Verde
    • Rol 3 - Amarillo
    • Rol 4 - Morado
    • Rol 5 - Rosa
  • Icono de Arrastre: Cada rol muestra un icono de líneas verticales que indica que es arrastrable.
  • Instrucción: "Arrastra un rol hacia un usuario para asignarlo" - Texto guía sobre cómo usar la funcionalidad.

3. Panel Derecho - Lista de Usuarios

Este panel muestra todos los usuarios del sistema:

  • Tarjetas de Usuario: Cada usuario aparece en una tarjeta individual con su nombre.
  • Badges de Rol: Si un usuario tiene un rol asignado, aparece como una etiqueta de color junto a su nombre.
  • Scroll Vertical: Si hay muchos usuarios, el panel permite desplazamiento vertical para ver todos.
  • Instrucción: "Arrastra un rol fuera del usuario para eliminarlo" - Texto guía sobre cómo remover roles.

Cómo Asignar Roles a Usuarios

Asignar un rol a un usuario es un proceso simple e intuitivo mediante arrastrar y soltar:

1

Seleccionar el Rol

En el panel izquierdo, haz clic y mantén presionado sobre el rol que deseas asignar.

Tip: El cursor cambiará a un icono de "mano agarrando" cuando estés arrastrando.
2

Arrastrar sobre el Usuario

Sin soltar el clic, arrastra el rol hacia la tarjeta del usuario al que deseas asignarlo.

Efecto Visual: La tarjeta del usuario se resaltará con un fondo azul claro y aumentará ligeramente de tamaño cuando el rol esté sobre ella.
3

Soltar el Rol

Suelta el clic cuando estés sobre el usuario deseado.

Confirmación: Aparecerá una notificación verde en la esquina superior derecha confirmando que el rol fue asignado exitosamente.
4

Verificar la Asignación

El rol asignado aparecerá inmediatamente como una etiqueta de color junto al nombre del usuario.

Importante

  • • Un usuario solo puede tener un rol asignado a la vez.
  • • Si asignas un nuevo rol a un usuario que ya tiene uno, el rol anterior será reemplazado automáticamente.
  • • Los cambios se guardan inmediatamente en la base de datos.

Cómo Eliminar Roles de Usuarios

Eliminar un rol asignado es igual de sencillo que asignarlo:

1

Seleccionar el Badge del Rol

Haz clic y mantén presionado sobre la etiqueta de rol que aparece junto al nombre del usuario.

Identificación: El badge del rol muestra el icono de líneas verticales indicando que es arrastrable.
2

Arrastrar Fuera del Usuario

Arrastra el badge del rol hacia cualquier área fuera de la lista de usuarios (puede ser hacia el panel de roles, hacia arriba, o hacia los costados).

Área Válida: Cualquier zona que no sea otra tarjeta de usuario es válida para eliminar el rol.
3

Soltar el Badge

Suelta el clic cuando estés fuera del área de usuarios.

Confirmación: Aparecerá una notificación confirmando que el rol fue eliminado exitosamente.
4

Verificar la Eliminación

El badge del rol desaparecerá inmediatamente de la tarjeta del usuario, dejándolo sin rol asignado.

Nota Importante

La eliminación del rol es inmediata y sin confirmación. El cambio se guarda automáticamente en la base de datos. Si eliminas un rol por error, simplemente vuelve a asignarlo siguiendo los pasos de asignación.

Consejos y Mejores Prácticas

Planifica Antes de Asignar

Define claramente qué permisos corresponden a cada rol antes de comenzar a asignarlos a los usuarios.

Verifica las Asignaciones

Después de asignar roles, verifica que los usuarios tengan los permisos correctos probando el acceso a diferentes módulos.

Principio de Mínimo Privilegio

Asigna a cada usuario solo el rol necesario para realizar sus funciones, evitando otorgar permisos excesivos.

Revisa Periódicamente

Realiza auditorías periódicas de los roles asignados para asegurar que sigan siendo apropiados conforme cambian las responsabilidades.

Protege el Rol de Administrador

Limita el número de usuarios con Rol 0 (Administrador) solo a personal de confianza y necesario.

Documenta los Roles

Mantén documentación actualizada sobre qué permisos y accesos otorga cada rol en el sistema.

Preguntas Frecuentes (FAQ)

¿Qué sucede si arrastro un rol sobre un usuario que ya tiene otro rol?

El rol anterior será reemplazado automáticamente por el nuevo rol. Un usuario solo puede tener un rol asignado a la vez.

¿Los cambios se guardan automáticamente?

Sí, todos los cambios (asignaciones y eliminaciones de roles) se guardan inmediatamente en la base de datos sin necesidad de hacer clic en un botón "Guardar".

¿Puedo deshacer una asignación de rol?

No hay un botón de "deshacer", pero puedes eliminar el rol inmediatamente arrastrándolo fuera del usuario, o asignar un rol diferente si fue un error.

¿Qué significa cada rol (Rol 1, Rol 2, etc.)?

Los nombres de los roles son genéricos en la interfaz. Consulta la documentación de tu sistema o con el administrador principal para conocer los permisos específicos de cada rol.

¿Por qué no puedo acceder al módulo de asignación de roles?

Solo los usuarios con Rol 0 (Administrador) tienen acceso a este módulo. Si necesitas acceso, contacta a un administrador del sistema.

¿Puedo asignar múltiples roles a un mismo usuario?

No, el sistema está diseñado para que cada usuario tenga únicamente un rol asignado a la vez. Esto simplifica la gestión de permisos y evita conflictos.

¿Qué hago si aparece un error al asignar o eliminar un rol?

Si aparece una notificación de error (en rojo), verifica tu conexión a internet y vuelve a intentarlo. Si el problema persiste, contacta al soporte técnico con el mensaje de error específico.

¿Los usuarios son notificados cuando se les asigna o cambia un rol?

El sistema no envía notificaciones automáticas. Es recomendable informar manualmente a los usuarios sobre cambios en sus roles y los nuevos permisos que esto implica.

¿Funciona en dispositivos móviles o tablets?

La interfaz está optimizada principalmente para uso en computadoras de escritorio. En dispositivos táctiles, la funcionalidad de arrastrar y soltar puede variar según el navegador.
