Programación de Mensajes

Comunicación efectiva en pantalla

Este módulo permite enviar mensajes instantáneos o programados que aparecerán directamente en la pantalla de los usuarios seleccionados. Es una herramienta ideal para anuncios importantes, recordatorios o avisos urgentes que requieren atención inmediata.

Selección Múltiple

Elija uno, varios o todos los usuarios activos del sistema como destinatarios del mensaje.

Programación Flexible

Envíe mensajes ahora mismo o prográmelos para una fecha y hora específica futura.

Guía Paso a Paso

1

Seleccionar Destinatarios

En el panel izquierdo, encontrará la lista de todos los usuarios disponibles.

  • Marque la casilla junto a cada usuario individualmente.
  • Use la casilla "Todos" en la parte superior para seleccionar a todos los usuarios.
  • Use el cuadro de búsqueda para filtrar la lista por nombre.
Debe seleccionar al menos un usuario para poder enviar el mensaje.
2

Configurar Fecha y Hora

En el formulario de la derecha, configure cuándo se debe mostrar el mensaje.

Por defecto, la fecha y hora están configuradas en el momento actual para envío inmediato.
3

Redactar y Enviar

Complete los detalles del mensaje:

  • Título: Un encabezado breve y claro (ej: "Reunión de Equipo").
  • Mensaje: El contenido detallado que verán los usuarios.

Finalmente, presione el botón "Enviar Mensaje(s)". Verá una confirmación si el proceso fue exitoso.
