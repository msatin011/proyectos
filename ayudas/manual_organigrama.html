Módulo de Organigramas

Este módulo permite diseñar y gestionar la estructura jerárquica de su organización de forma visual e intuitiva, vinculando colaboradores a cada nivel de responsabilidad.

Estructura Dinámica

Cree niveles y subniveles sin límites de profundidad.

Asignación Visual

Asigne colaboradores a cada nivel mediante un selector interactivo.

1. Gestión de Organigramas

Al ingresar al módulo, verá una grilla con los organigramas creados. Desde el toolbar superior puede:

1

Crear:

Inicia un nuevo organigrama desde cero.

2

Modificar:

Seleccione una fila y haga clic para abrir el editor visual.

3

Eliminar:

Elimina el organigrama seleccionado (se requiere confirmación).

2. Editor Visual

El editor de pantalla completa permite construir el árbol jerárquico:

  • Botón Azul (+): Crea un nuevo subnivel dependiente del nivel actual.
  • Botón Verde (Usuario): Abre el panel de asignación de colaboradores.
  • Botón Rojo (Tacho): Elimina el nivel y todos sus descendientes.
  • Renombrar: Haga clic en el nombre del nivel para editarlo.

Tip de Estabilidad

Los cambios en el dibujo se mantienen temporalmente. Debe usar el botón Grabar del editor para persistirlos en la base de datos.

3. Asignación de Colaboradores

Al hacer clic en el icono de usuario de un nivel, se despliega un pop-up:

  • ✅ Indica que el colaborador está asignado al nivel.
  • ▢ Indica que el colaborador no está asignado.
  • Use el encabezado "Seleccionar Todos" para marcar/desmarcar toda la lista.
